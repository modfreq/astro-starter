## Text Formatting

You can write **bold text**, _italic text_, and **_bold italic text_**. Use `inline code` for technical terms and ~~strikethrough~~ for corrections.

## Code Blocks

Inline code looks like `const x = 42`, and code blocks look like this:

```typescript
import { defineCollection, z } from "astro:content";

const blog = defineCollection({
  schema: z.object({
    title: z.string(),
    date: z.coerce.date(),
  }),
});

export const collections = { blog };
```
